GIT шпаргалка

Base

  • git config --global user.name blabla - задает имя, передаваемое в коммитах.
  • git config --global user.email blabal@bla - задает email, передаваемый в коммитах.
  • git init - команда создаёт в текущем каталоге новый подкаталог с именем .git, содержащий все необходимые файлы репозитория — структуру Git репозитория.
  • git status - отображает состояние рабочего каталога и раздела проиндексированных файлов.
  • git add - добавляет содержимое рабочего каталога в staging area для последующего коммита.
  • git commit - команда для записи индексированных изменений в репозиторий.
  • git log - перечисляет коммиты, сделанные в репозитории в обратном к хронологическому порядке.
  • git checkout - используется для переключения на существующую ветку.
  • git branch - команда для управления ветками в репозитории Git.
  • git push - устанавливает связь с удалённым репозиторием, вычисляют локальные изменения отсутствующие в нём, и передает их в вышеупомянутый репозиторий.
  • git pull - забирает изменения из указанного удалённого репозитория, а затем пытается слить их с текущей веткой.
  • git merge - используется для слияния одной или нескольких веток в текущую.
  • git clone - это утилита для выбора существующего репозитория и создания его клона.

.gitignore - нужен для скрытия файлов и папок от системы контроля версий Git. Обычно скрывают конфигурационные файлы (особенно с паролями), временные файли и папки.

Advanced

  • git revert - откатывает коммит путём создания нового, изменения в котором обратны тому коммиту, который нужно отменить.
  • git reset - отменяет изменения в разделе проиндексированных файлов (используется с аргуметами --soft, --mixed, --hard).
  • git rebase - объединяет изменения, сделанные в одной ветке, с другой веткой.
  • git cherry-pick - применяет коммиты из одной ветви в другую, но не сохраняет исходную историю.
  • git stash - сохраняет отслеживаемые файлы и проиндексированные изменения в хранилище незавершённых изменений, которые вы можете в любое время применить обратно.
